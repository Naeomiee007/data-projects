# Airbnb Market Analysis: NYC vs Los Angeles

SQL and Python project analyzing Airbnb listings to answer a key question:

**Is Airbnb dominated by individual hosts or commercial property managers?**

Dataset: Inside Airbnb public data  
Markets analyzed: **New York City and Los Angeles**  
Total listings analyzed: **81,000+**

## Tech Used

- SQLite
- Python
- Pandas
- Matplotlib / Seaborn

## Key Insight

A small number of hosts control extremely large portfolios.

Some hosts manage **500–1,000+ listings**, indicating strong platform commercialization rather than casual home sharing.
