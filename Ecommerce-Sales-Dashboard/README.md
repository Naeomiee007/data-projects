# Ecommerce Sales & Customer Insights Dashboard

Interactive Tableau dashboard analyzing ecommerce transaction data to identify revenue trends, top customers, and best-selling products.

## Tools

Python (data preparation)
Tableau Public (dashboard & visualization)
GitHub (project documentation)

## Dashboard Features

* KPI metrics: Total Revenue, Orders, Customers
* Sales trend over time
* Revenue by country
* Top performing products
* Top revenue generating customers

## Interactivity

The dashboard allows users to explore the data dynamically:

* Click a country to filter other charts
* Hover for detailed tooltips
* Interactive exploration of customers, products, and revenue trends

## Key Insights

* Revenue is concentrated in a few major markets
* A small group of customers drives a large portion of sales
* Certain products dominate revenue performance
* Revenue trends show seasonal patterns
